The Runkeeper Fitness Feed adds a widget which displays up to your last 20 activities.

== Description ==

The Runkeeper WordPress Widget allows you to display up to the last 20 Runkeeper activities 
with the widget area in your WordPress blog. It displays:

- Activity type
- Distance (km)
- Duration (h:m:s)

You can customise the following:

- Feed title
- Feed length (1, 5, 10, 15, 20)
- Display size (small, medium, large)

== Installation ==

1. Download the latest version
2. Copy it into the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Authorise the use of the widget with Runkeeper by visiting http://pragmaticrunner.com/code/runkeeper-wordpress-widget/
4. Copy the code that was sent via email
5. Paste the code into the 'Auth Code' field (Appearance > Widgets > Runkeeper Feed)
5. (Optional) Customize the plugin Title, Feed length and Display size.
